# Java多线程下载器-[JDownloader](https://github.com/yoyling/JDownloader)

## 开发环境
* JDK 8 + IDEA 2020.3

## 功能
* - [x] 支持多线程
* - [x] 监听剪贴板
* - [x] 基于流的传输

## 案例截图

一个 776 MB 的 OneDrive 视频文件。

正常谷歌浏览器下载需要 7分20秒，平均 1.76M/s；多线程需要90秒，平均 8.6M/s，5倍下载效率；
